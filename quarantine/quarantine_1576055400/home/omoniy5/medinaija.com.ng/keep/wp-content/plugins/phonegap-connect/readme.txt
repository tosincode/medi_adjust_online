Plugin Phonegap - Connect Plugin for Phonegap App
== Description ==

Plugin para que te provee de una conexión directa con Wordpress desde tu aplicación Phonegap, la librería de conexión y su documentación está disponible en Github en <a href="https://github.com/DanielRiera/phonegap-wordpress">este enlace</a>.

Si necesitas algún tipo de soporte dirigete a mi blog en <a href="http://www.danielriera.net" target="_blank">este enlace</a>



== Installation ==

Aquí se detalla como instalar y poner en funcionamiento el sistema

e.g.

1. Instalar el plugin desde wordpress.org/plugins o también puedes descargar el plugin y subirlo directamente al directorio /wp-content/plugins/.
2. Activar el plugin
3. En el menú lateral se ha creado un nuevo botón para acceder a la configuración del plugin, copia el token generado a través de la librería
